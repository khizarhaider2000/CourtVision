
# NBA AI Stats Analyzer

An NBA analytics tool that allows users to request statistical visualizations in plain English.  
The system converts natural-language queries into structured chart specifications and generates correct, explainable NBA stat visualizations.

## Features (v1)
- Team-level NBA analytics
- Offensive Rating, Defensive Rating, Net Rating, Pace, eFG%, TS%
- Time windows: season-to-date, last 5/10/20 games
- Leaderboard and efficiency landscape visualizations
- Deterministic metric computation from canonical datasets

## Tech Stack
- Python
- pandas, numpy
- matplotlib
- nba_api
